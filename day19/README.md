question of the day: https://leetcode.com/problems/01-matrix/#/description

Given a matrix consists of 0 and 1, find the distance of
the nearest 0 for each cell.

The distance between two adjacent cells is 1.

*Example 1:*

Input:

```ruby
0 0 0
0 1 0
0 0 0
```

Output:

```ruby
0 0 0
0 1 0
0 0 0
```

*Example 2:*

Input:

```ruby
0 0 0
0 1 0
1 1 1
```

Output:

```ruby
0 0 0
0 1 0
1 2 1
```
